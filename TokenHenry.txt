1.-Primero que nada debemos de tenes una URL al cual vamos a clonar nuesro repositorio, la clonaremos con nuestra terminal 
  GIT dando clik derecho y selecionando "Git Bash Here" una vez que nos abra la terminal con el comando:
  "[git clone (URL del repositorio que deseamos clonar)]" Y damos enter.

2.- Una vez clonado el repositorio lo abrimos con Visual Code, modificamos lo que tengamos que modificar, damos "Save"
   y depues tendremos que ejecutar los siguente comandos para guardar y sincronizar los cambios en nuestro repositorio
   con el repositorio remoto que esta en GitHub.
   
3.- Abrimos la terminal "ctrl + Ã±" vemos el estatus del codigo y sus cambios: "git status" ejecutando este comando
    nos debera aparecer en rojo la linea de codigo del archivo modificado.

4.- Despues lo tenemos que agregar con el comando: "git add" damos un estatus "git status" y nos tendra que aparecer 
    en verde, que quiere decier que se agrego al historial de Github correctamente.

5.-Ahora tendremos que verificar esos cambios con el comando: "[git commit -m  "cualquier descripcion"] " una vez confirmados 
   los cambios tendremo que subir los cambios al repositorio de Github que es un servidor.
	.si te salta un error de que no encuentra la direccion email, solo debemos establecer nuestro email con y nuestro 
	.nombre de usuario con el sigieinte comamdo GIT: [git config --global user.email (nuestro email)]
							 [git config --global user.name (Nuestro nombre)]

6.- Para subir los cambios al repositorio de GitHub usamos el siguiente comando: "[git push]" en donde nos pedira por primera
    vez una autenticacion para eso tendremos que crear un tocken.
